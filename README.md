### What I learned

- How to use grid column gap and grid row gap.
- How to use grid row.
- How to use box-shadow.

```css
  .container{
    grid-row: 2/3;
    grid-column-gap: 30px;
    grid-row-gap: 5px;
  }

  .cards{
    box-shadow: -1px 2px 13px var(--greyish-blue);
  }
```
